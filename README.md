# 🎮 FlowPlay

Aplicativo mobile desenvolvido em Capacitor + Android com integração de frontend e backend.

## Tecnologias utilizadas
- CapacitorJS - Bridge entre Web e Native
- Android (Java + Gradle) - Plataforma nativa
- Node.js - Ambiente de desenvolvimento
- Git/GitHub - Versionamento
- ImageMagick - Processamento de ícones

## Estrutura do projeto
- android/ - Plataforma Android
- www/ - Frontend web
- resources/ - Assets originais
- icons/ - Ícones PWA

## Assets Gerados
Ícones adaptativos e splash screens para todas as densidades Android, modo escuro e PWA.

## Comandos úteis
npm install
npx cap sync android
cd android && ./gradlew installDebug
npx @capacitor/assets generate
